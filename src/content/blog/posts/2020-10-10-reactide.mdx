### Reactide is the first dedicated IDE for React web application development.

Reactide is a cross-platform desktop application that offers a simulator, made for live reloading and quick React component prototyping. React brings an integrated suite of development tools to streamline react development. The days of flipping between browser, IDE, and server are over. If you want to know more about Reactide visit [main page](http://reactide.io/) and [GitHub](https://github.com/reactide/reactide)
